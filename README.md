# Lista de Exercícios - Semana 11 e 12 - Bootcamp Data Analytics Womakers <img width="48" height="48" src="https://img.icons8.com/emoji/48/butterfly-emoji.png" alt="butterfly-emoji"/>

Este repositório contém uma lista de exercícios desenvolvida durante a 11ª e 12ª semana do Bootcamp em Data Analytics do projeto Womakers. O foco desta semana foi em Estatística com Python, abordando os conceitos de Probabilidade e Amostragem.

## Descrição

Durante duas semana do Bootcamp em Data Analytics, exploramos os seguintes tópicos:

- Conceitos fundamentais de Estatística, incluindo eventos, espaços amostrais de probabilidade.
- Cálculo de probabilidades de eventos simples e compostos.
- Distribuições de probabilidade discretas, como a distribuição binomial e a distribuição de Poisson.
- Distribuições de probabilidade contínuas, como a distribuição normal e a distribuição exponencial.
- Técnicas de amostragem, incluindo amostragem aleatória simples e amostragem estratificada.
- Principais bibliotecas Python utilizadas para estatística

Este repositório contém uma série de exercícios práticos que visam consolidar os conhecimentos adquiridos ao longo da semana.

Este projeto foi desenvolvido como parte do Bootcamp em Data Analytics da Womakers. Para saber mais sobre a Womakers e seus programas, visite [womakerscode.org](https://womakerscode.org/).
